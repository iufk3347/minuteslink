## 有什么坑吗?

老实说,没有完美的工具。MinutesLink也有几个小缺点:

1. **目前只支持Zoom和Google Meet**:如果你们公司用的是Microsoft Teams或者其他平台,暂时用不了(不过官方说Teams支持在路上了)。

2. **AI处理次数有限**:免费版每月10次,如果你会议特别多,可能不够用。不过你可以选择只对重要会议开启AI处理,日常站会就不用了。

3. **偶尔会有小错误**:比如有人名字比较生僻,或者口音太重,AI可能会转录错。不过这种情况很少,而且可以手动修改。

4. **网页上找不到优惠码**:我搜遍了全网,也没找到MinutesLink的promo code或者discount code。看来他们走的是"价格透明,童叟无欺"路线,没那么多营销套路。

不过这些都是小问题,不影响整体体验。
